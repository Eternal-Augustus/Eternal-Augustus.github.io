要点：

1，抽帧,重命名 2，npy数据转换成npz 数据

抽帧

将拍摄的视频抽帧成不同视角下的图片

使用 extract_frames.py

cd C:\Users\71905\Desktop\LLFF-master\video
python extract_frames.py --video_in "20240720_183328_474.mp4" --video_fps 5

或者

python extract_frames.py --video_in "C:\Users\71905\Desktop\LLFF-master\video\20240720_183328_474.mp4" --video_fps 5

Colmap 稀疏重建 

File -> New project
Processing -> Feature extraction->Camera model ->SIMPLE_PINHOLE
Processing -> Feature matching
Reconstruction -> Start reconstruction
File -> Export model /sparse/0/

注意:匹配的位姿和图片数目不同在后续的步骤中会导致报错

生成LLFF数据

LLFF-master目录下，打开imgs2poses.py

python imgs2poses.py "XXXX/XXXX/"

遇到位姿和图片不匹配的问题，删掉不符合要求的图片重新进行稀疏重建 得到npy文件
